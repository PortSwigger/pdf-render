This extension enables rendering of PDF files directly within HTTP responses. When a response contains a PDF document, it is displayed in a dedicated tab, making it easy to inspect content without leaving the Burp Suite interface.

Features

  • Automatically detects and renders PDF files in HTTP responses
  • Displays rendered PDF content in a custom response tab
  • Supports interactive viewing and scrolling within the embedded PDF

Usage

  1. Browse or intercept traffic containing PDF responses.
  2. Select a response containing a PDF file.
  3. Navigate to the custom PDF tab in the response viewer to see the rendered document.
